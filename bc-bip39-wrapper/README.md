# Blockchain Commons BIP39 wrapper library for Java/Android

# Prerequisites
- Java version 1.8+
- Android API level 21+

# Installation
###Gradle

`implementation 'com.bitmark.sdk:bc-bip39-wrapper:1.0.0-SNAPSHOT'`

# For maintainers

### Build
`cd scripts/`

`./build-mac.sh` (on Mac)

or 

`./build-linux.sh` (on Linux)

### Deploy


#####SNAPSHOT

`gradle publish --info`

#####Release

`gradle bintrayUpload --info`
