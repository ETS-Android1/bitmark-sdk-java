# Bitmark SDK for Java
The official Bitmark SDK for Java

## Setting Up

### Prerequisites

- Java version 1.8+

### Installing

##### Gradle
```sh
dependencies {
    implementation 'com.bitmark.sdk:java-sdk:2.1.0'
}
```

##### Maven
```sh
<dependency>
    <groupId>com.bitmark.sdk</groupId>
    <artifactId>java-sdk</artifactId>
    <version>2.1.0</version>
</dependency>
```
